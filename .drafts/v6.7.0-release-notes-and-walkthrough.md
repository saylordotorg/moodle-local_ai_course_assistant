# SOLA v6.7.0 — Soapbox speech practice (with ESL levels), release notes + admin walkthrough

## Part 1: Release Notes (15 June 2026)

**v6.7.0 introduces Soapbox, an AI speech-coaching tool for speech and ESL courses. Off by default; enable per course.**

A learner optionally sets a name, topic, and target length, records a longer spoken presentation, which is transcribed and scored against a per-course speaking rubric with concrete tips. A personal score history is kept. Audio and transcript text are never stored, only the scores, feedback, and the name/topic/target you chose.

- **Soapbox learner page (`soapbox.php`).** Setup panel (name / topic / target length), record-and-stop, rubric-scored feedback, and a "My Speeches" history. Reached from a course-navigation link shown to enrolled learners when the feature is on for that course.
- **Three transcription tiers, chosen by the `soapbox_stt_mode` setting (server / browser):**
  - Self-hosted Whisper (free; reuses the v6.3.0 self-hosted STT path).
  - Hosted OpenAI Whisper (about USD 0.006/minute; no infrastructure).
  - In-browser Web Speech API (free, no server, no cost; Chrome and Safari).
  Server mode auto-selects self-hosted vs hosted via the voice registry.
- **ESL as well as speech courses.** A per-course "Course type / speaking level" setting picks one of three sample rubrics and tailors the AI coaching register:
  - **General speech / presentation** (rhetoric, structure, delivery) — the default.
  - **ESL (beginner)** — pronunciation/intelligibility, fluency/pace, basic grammar, core vocabulary, task completion; gentle, encouraging feedback that does not penalise an accent.
  - **ESL (advanced)** — fluency/naturalness, stress/intonation, grammatical range, idiom/register, coherence; held to a high standard while staying encouraging.
- **Editable rubrics.** The new `speech` rubric type is fully authorable per course on the rubric admin page (a "Soapbox Speech" tab), with a global default fallback and a "Load a sample rubric" picker that seeds the editor with any of the three samples to review and Save.
- **Privacy.** A nullable `session_meta` column on the practice-scores table holds only the name/topic/target blob; never audio or transcript. Covered by the privacy provider (metadata + export + erasure).
- **Cost controls.** Off by default; per-course gated. Long-form STT endpoint has a 25 MB cap, a 300s timeout, and a dedicated rate bucket (12 per 10 minutes per learner). Browser STT mode is entirely free.
- **Full i18n.** All 50 new Soapbox strings (41 feature + 9 ESL/level UI) translated into all 45 non-English languages. 46/46 locales pass the completeness check with zero missing keys.

## Part 3: Admin walkthrough — setting up Soapbox for a course (about 8 minutes)

This is the new flow in v6.7.0. For the broader 20-minute SOLA walkthrough, see the prior release drafts.

1. **Turn the feature on for the course.** Open the course, then the **More** menu, **Course AI Settings**. Find **Soapbox** in the pedagogy toggles. Set it to **Force on** for this course (the site default is off). Save.
2. **Pick the course type / speaking level.** Once Soapbox is on, a **Course type / speaking level** dropdown appears: General speech / presentation, ESL (beginner), or ESL (advanced). For a speaking or communication course, leave it on General. For an English-language course, pick the matching ESL level. This sets the coaching tone and the default rubric.
3. **(Optional) Tune the rubric.** Click **Edit speech rubric**. On the "Soapbox Speech" tab, use **Load a sample rubric** to drop in General / ESL beginner / ESL advanced criteria, edit names, descriptions, and max scores to taste, then **Save Rubric**. Saving for a specific course creates a course-specific override; the global default covers any course you do not customize.
4. **Choose how speech is transcribed.** In Site administration, the **Soapbox transcription mode** setting picks Server (Whisper provider) or Browser (free, no server). Server is recommended when you have a self-hosted Whisper server (free) or accept hosted OpenAI Whisper (about USD 0.006/minute). Browser mode needs no server and costs nothing, but works only in Chrome and Safari.
5. **Try it as a learner.** Open the course as an enrolled student. A **Soapbox speech practice** link appears in the course navigation. Set an optional name, topic, and target length, record a short speech, stop, and read the rubric-scored feedback and tips. Confirm the feedback register matches the level you chose. The new attempt appears under **My speeches**.
6. **What is and is not stored.** Reassure faculty: the score, the feedback, the duration, and the optional name/topic/target are saved to the learner's history. The audio recording and the transcript text are never stored.

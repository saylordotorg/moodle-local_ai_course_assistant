# SOLA v5.11.1 — Release Notes (9 June 2026)

CI-only hotfix on top of v5.11.0. Cherry-picks the docblock fix that landed on main as commit a80644b, so the `v5.11.1` tag passes `moodle-plugin-ci phpdoc --max-warnings 0`. Zero behaviour change vs v5.11.0.

- **Fix:** `admin/cli/run_tutor_golden.php` — `mode_run()` and `mode_judge()` were missing the `@param string $promptsfile` line in their docblocks (the parameter itself was added in v5.11.0 for the `--prompts=FILE` flag used by the A.10 bake-off). PHPDoc Checker required it; failing run was 27246452269.

### Backwards compatibility

`$plugin->version` is unchanged at `2026060900` (same as v5.11.0). A site running v5.11.0 will not detect this as an upgrade (correct: there is no code-semantic change, only a 2-line docblock). The hotfix exists purely so the tagged release passes CI for downstream consumers who gate on tag-level CI status.

If you are already on v5.12.0, install v5.12.1 instead (same docblock fix on top of v5.12.0 content).
